# 3×3 LED Cube using Arduino Nano

![3x3 LED Cube](led.png)

## Project Overview
This project is a 3×3 LED cube controlled using an Arduino Nano. The system demonstrates digital output control, pattern generation, and timing logic in embedded systems.

## Description
The cube consists of 9 LEDs arranged in a 3×3 structure, each connected to Arduino digital output pins. By controlling the ON/OFF states of the LEDs in specific sequences, different visual patterns and animations are generated.

The project focuses on understanding how software interacts with hardware to produce controlled visual effects.

## Hardware & Tools
- Microcontroller: Arduino Nano  
- LEDs: 9 LEDs arranged in 3×3 structure  
- Resistors: Current limiting resistors  
- Jumper wires / soldered connections  
- Breadboard or custom structure  
- Programming Language: Arduino (C/C++)  
- IDE: Arduino IDE  
- Power Supply: USB / external battery  

## Working Principle
- Each LED is connected to a digital output pin of the Arduino  
- LEDs are controlled using HIGH (ON) and LOW (OFF) signals  
- Patterns are generated by activating LEDs in specific sequences  
- Delay functions control the timing of animations  
- Loop structures enable continuous pattern repetition

- LED wiring order directly affects animation behavior  
- Proper resistor selection is essential to prevent LED damage

- Delay timing controls animation speed and visibility  
- Structuring patterns into functions improves modularity
